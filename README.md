## 📘 1.3 作业
每个作业留下来的时间是两周

| 作业  | PPT               | 时间区间  | 备注|
|:-------:|:-------------------:|:-----------:|:---|
| 作业一 | lecture1~lecture4 | 4.11~4.20 |其实作业一有部分内容是lecture6里的，所以建议看完6再去进行作业一，会更得心应手一些|
| 作业二 | lecture5~lecture9 | 4.20~5.4 | 没有用到深度学习框架，作业内容其实只涉及到lecture7|
